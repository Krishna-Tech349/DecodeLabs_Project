 ByteBot - Rule-Based AI Chatbot

## About
A rule-Based chatbot built as part of DecodeLabs AI Internship.

## Features
- 25+ conversation rules
- Remembers user name
- Saves chat log to file
- Handles empty inputs
- Random fallback responses
- Real time and date responses

## How to Run
1. Install Python
2. Run: python chatbot.py

## Tech Used
- Python 3
- Built-in libraries only (random, datetime)
